# Log Analyzer 

![Python Version](https://img.shields.io/badge/python-3.7+-blue.svg)
![Status](https://img.shields.io/badge/status-active-success.svg)

Простой, но мощный инструмент для анализа логов различных систем на Python. Поддерживает парсинг и анализ логов Apache, Nginx, системных логов (syslog) и других форматов.

## Особенности

- **Мультиформатный парсинг**: Apache, Nginx, Syslog
- **Умная фильтрация**: по ключевым словам, уровню, типу
- **Детальная статистика**: ошибки, предупреждения, распределение
- **Автоматическое определение**: типа логов
- **Интерактивный интерфейс**: удобное меню управления
- **Тестовые данные**: для быстрого старта

## Использование

Интерфейс командной строки

После запуска программы вы увидите интерактивное меню с опциями:

1. Показать все логи - отображение первых 20 записей

2. Фильтровать логи - продвинутая фильтрация по критериям

3. Показать статистику - детальный анализ и отчет

4. Поиск по ключевому слову - быстрый поиск

Выход - завершение работы

## Примеры команд
```bash
# Использование в скриптах
from log_parser import parse_log_file
from log_stats import generate_report

# Парсинг файла
logs = parse_log_file('/var/log/apache2/access.log', log_type='apache')

# Генерация отчета
report = generate_report(logs)
print(f"Найдено ошибок: {report['errors']}")
```
